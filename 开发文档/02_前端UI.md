太棒了！既然“咖啡馆”已经成功开业并能吐出文字，我们接下来的目标是把这个简陋的文字页面变成一个**真正的管理面板界面**。

根据之前的技术文档，我们现在的任务是 **阶段四：前端 UI (HTML 模板)**。我们将让 Go 加载一个 HTML 文件，而不是直接发送字符串。

---

## 本阶段任务：实现 HTML 页面渲染

### 1. 准备文件夹结构
Go 不会自动乱找文件，你需要告诉它模板在哪。在你的 `BOBO` 文件夹下创建两个新文件夹：
* **`templates`**: 存放 HTML 模板文件。
* **`static`**: 存放 CSS、图片等静态资源（虽然我们现在先不用，但建议留好位置）。

你的目录现在应该是：
```text
BOBO/
├── go.mod
├── main.go
├── templates/
└── static/
```

---

### 2. 创建 HTML 模板
在 `templates` 文件夹下创建一个文件：**`index.html`**。
为了让界面立刻变好看，我们引入 **Pico.css**，这样你不需要写任何 CSS 就能得到一个深色模式、响应式的优雅界面。

将以下内容填入 `templates/index.html`:

```html
<!DOCTYPE html>
<html lang="zh">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Bobo Dev 面板</title>
    <link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/@picocss/pico@1/css/pico.min.css">
</head>
<body>
    <main class="container">
        <nav>
            <ul>
                <li><strong>🚀 Bobo Dev Project Manager</strong></li>
            </ul>
        </nav>

        <section>
            <h2>我的项目</h2>
            <article>
                <header>示例项目 A</header>
                <p>路径: <code>D:/Github_Projects/Test</code></p>
                <p>备注: 这是一个测试项目</p>
                <footer>
                    <button>启动项目</button>
                    <button class="secondary">打开文件夹</button>
                </footer>
            </article>
        </section>

        <section>
            <details>
                <summary>+ 添加新项目</summary>
                <form>
                    <input type="text" name="name" placeholder="项目名称" required>
                    <input type="text" name="path" placeholder="项目路径 (如 D:/Work/App)" required>
                    <input type="text" name="cmd" placeholder="启动指令 (如 python main.py)">
                    <button type="submit">保存项目</button>
                </form>
            </details>
        </section>
    </main>
</body>
</html>
```

---

### 3. 修改 main.go 以加载模板
现在我们需要修改 Go 代码，让它去读取这个 HTML 文件并渲染出来。



修改你的 `main.go` 如下：

```go
package main

import (
	"github.com/gin-gonic/gin"
	"net/http"
)

func main() {
	r := gin.Default()

	// 1. 告诉 Gin 你的模板文件在哪个文件夹
	r.LoadHTMLGlob("templates/*")

	// 2. 修改路由逻辑：从 c.String 变成 c.HTML
	r.GET("/", func(c *gin.Context) {
		// 返回 HTML 页面
		// "index.html" 对应 templates 文件夹下的文件名
		// gin.H 是一个 Map，用来给前端传数据，现在我们先传个标题
		c.HTML(http.StatusOK, "index.html", gin.H{
			"Title": "Bobo Dev",
		})
	})

	r.Run(":5051")
}
```

---

### 4. 运行并查看
1.  在终端按 `Ctrl + C` 停止之前运行的程序。
2.  输入 `go run main.go`。
3.  刷新浏览器 `http://localhost:5051`。

**你现在应该能看到一个带有深色背景、精致边框和按钮的管理面板雏形了！**

---

### 💡 为什么这样做？
* **LoadHTMLGlob**: 预加载所有模板，这样速度极快。
* **gin.H**: 这是一个“字典”（类似 Python 的 `dict`），你往里填什么，前端 HTML 就可以通过 `{{ .Title }}` 这种语法拿到什么。
