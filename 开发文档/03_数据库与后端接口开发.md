## 本阶段任务：对接 SQLite 数据库

### 1. 安装 SQLite 驱动
Go 语言操作 SQLite 需要一个驱动。在终端输入：
```bash
go get github.com/mattn/go-sqlite3
```
*注意：这个库需要 CGO 支持（即你之前安装的 GCC 编译器），如果报错，请检查 GCC 是否安装成功。*

---

### 2. 定义数据模型 (models.go)
为了让代码清晰，我们在 `BOBO` 文件夹下新建一个文件 `models.go`。它就像是一个“模具”，定义了每个项目在数据库里长什么样。

```go
package main

// Project 结构体对应数据库里的一行数据
type Project struct {
	ID      int    `json:"id" gorm:"primaryKey"`
	Name    string `json:"name"`
	Path    string `json:"path"`
	Command string `json:"command"`
	Note    string `json:"note"`
}
```

---

### 3. 初始化数据库 (database.go)
新建 `database.go`，负责创建数据库文件并建表。

```go
package main

import (
	"database/sql"
	_ "github.com/mattn/go-sqlite3" // 隐式导入驱动
	"log"
)

var DB *sql.DB

func InitDB() {
	var err error
	// 打开（或创建）名为 bobo.db 的文件
	DB, err = sql.Open("sqlite3", "./bobo.db")
	if err != nil {
		log.Fatal(err)
	}

	// 创建表的 SQL 语句
	sqlStmt := `
	CREATE TABLE IF NOT EXISTS projects (
		id INTEGER PRIMARY KEY AUTOINCREMENT,
		name TEXT,
		path TEXT,
		command TEXT,
		note TEXT
	);`

	_, err = DB.Exec(sqlStmt)
	if err != nil {
		log.Printf("%q: %s\n", err, sqlStmt)
		return
	}
}
```

---

### 4. 串联到主程序 (main.go)
现在修改 `main.go`，让它在启动时初始化数据库，并能把数据库里的项目传给前端。



```go
package main

import (
	"github.com/gin-gonic/gin"
	"net/http"
)

func main() {
	// 1. 初始化数据库
	InitDB()
	defer DB.Close() // 程序关闭时释放数据库连接

	r := gin.Default()
	r.LoadHTMLGlob("templates/*")

	// 2. 首页路由：从数据库读取项目
	r.GET("/", func(c *gin.Context) {
		rows, _ := DB.Query("SELECT id, name, path, command, note FROM projects")
		var projects []Project

		for rows.Next() {
			var p Project
			rows.Scan(&p.ID, &p.Name, &p.Path, &p.Command, &p.Note)
			projects = append(projects, p)
		}

		c.HTML(http.StatusOK, "index.html", gin.H{
			"Projects": projects,
		})
	})

	// 3. 接收表单数据的路由（新增项目）
	r.POST("/add", func(c *gin.Context) {
		name := c.PostForm("name")
		path := c.PostForm("path")
		cmd := c.PostForm("cmd")
		note := c.PostForm("note")

		_, err := DB.Exec("INSERT INTO projects (name, path, command, note) VALUES (?, ?, ?, ?)", 
			name, path, cmd, note)
		
		if err != nil {
			c.String(http.StatusInternalServerError, "保存失败")
			return
		}
		// 保存成功后重定向回首页
		c.Redirect(http.StatusMovedPermanently, "/")
	})

	r.Run(":5051")
}
```

### 5. 最后一步：修改 HTML 提交地址
打开 `templates/index.html`，找到 `<form>` 标签，修改如下：

```html

    <input type="text" name="cmd" placeholder="启动指令">
```
