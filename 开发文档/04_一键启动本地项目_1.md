## 本阶段任务：实现“一键启动”功能

我们将通过 Go 的 `os/exec` 包来调用系统的终端命令。

### 1. 核心逻辑原理
当你在网页点击“启动”按钮时：
1. 前端发送该项目的 `ID` 给后端。
2. 后端去数据库查出该项目的 `Path`（路径）和 `Command`（指令）。
3. 后端启动一个**子进程**去运行这个指令。



---

### 2. 在 `main.go` 中添加启动接口

**更新：这部分查看"04_一键启动本地项目_2.md"**

我们需要新增一个路由来处理启动请求。

```go
package main

import (
	"github.com/gin-gonic/gin"
	"net/http"
	"os/exec" // 必须导入这个包，用来执行系统命令
	"runtime" // 用来判断是 Windows 还是 Mac/Linux
)

// 在 main 函数里添加这个路由
func setupRoutes(r *gin.Engine) {
    // ...之前的路由保持不变...

    r.GET("/run/:id", func(c *gin.Context) {
        id := c.Param("id")
        var p Project
        
        // 1. 从数据库找到这个项目
        err := DB.QueryRow("SELECT path, command FROM projects WHERE id = ?", id).Scan(&p.Path, &p.Command)
        if err != nil {
            c.String(http.StatusNotFound, "找不到该项目")
            return
        }

        // 2. 根据系统准备命令
        var cmd *exec.Cmd
        if runtime.GOOS == "windows" {
            // Windows 下通过 cmd /c 运行
            cmd = exec.Command("cmd", "/C", p.Command)
        } else {
            // Mac/Linux 下通过 sh -c 运行
            cmd = exec.Command("sh", "-c", p.Command)
        }

        // 3. 设置命令运行的目录（非常重要！）
        cmd.Dir = p.Path

        // 4. 异步启动命令（不要用 cmd.Run()，否则网页会一直转圈卡住）
        err = cmd.Start()
        if err != nil {
            c.String(http.StatusInternalServerError, "启动失败: "+err.Error())
            return
        }

        // 5. 启动成功，跳回首页
        c.Redirect(http.StatusMovedPermanently, "/")
    })
}
```

### 4. 进阶：如何实现 `bobo dev` 命令？
现在你每次都要进入 `BOBO` 文件夹输入 `go run .`，这不符合我们最初“bobo dev”的设计。

**我们需要做两件事：**

#### A. 编译成可执行文件
在终端输入：
```bash
go build -o bobo.exe
```
这会在文件夹下生成一个 `bobo.exe`。

#### B. 将文件夹加入环境变量 (Path)
1. 复制你 `BOBO` 文件夹的路径（例如 `D:\Github_Projects\BOBO`）。
2. 在 Windows 搜索框输入“环境变量”，打开“编辑系统环境变量”。
3. 在“用户变量”里找到 `Path`，点击“编辑” -> “新建”，把路径粘贴进去。
4. **重启终端**。

现在，你在**任何地方**输入 `bobo`（假设你把 main.go 里处理了命令行参数），它都能启动了。

```go
import "os"

func main() {
    // 检查命令行参数
    if len(os.Args) < 2 || os.Args[1] != "dev" {
        println("请使用 'bobo dev' 来启动服务")
        return
    }

    // 原有的初始化和启动代码...
    InitDB()
    // ...
}
```
